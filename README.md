# Android AI Handwritten Diary

A premium Android diary application that renders typed input as realistic handwritten text.

## Features
- Real-time handwritten rendering
- Paper texture with ruled pages
- Invisible keyboard input
- Flutter CustomPainter engine
- Web prototype for UI validation
